Introduction to Java

Java is one of the most popular coding languages worldwide. Normally, students take AP Computer Science as a sophomore to learn Java, and this is a bootcamp to get you familiarized to the language beforehand!

Prerequisites: None

Required Software: The ability to download an IDE (e.g. Eclipse) and Java JDK

Age/Grade: 4th Grade and Above

Cost: $50 (will be donated to local businesses)

By the end of this course, students will understand:

 -Understanding the Basics of the Java language
 -Variable and variable types
 -Arrays
 -For/While Loops
 -ArrayLists
 -Functions
 -Classes
 -Objects
 -Debugging skills and good coding habits
